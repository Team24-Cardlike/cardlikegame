<h1>Rouges of Midgard</h1>


<h2>Requirements:</h2>
<ul><li>Java 23+</li>
    <li>Maven 3.8+</li>
    <li>LibGDX 1.12.1+</li>
    <li>Gson 2.10.1+</li>
    <li>Junit 4.13.2+</li>
</ul>

<h2>Running the game:</h2>
<h3>Option 1:</h3>
Run `src/main/java/org/example/DesktopLauncher.java`.

<h3>Option 2:</h3>
Run the command `mvn compile exec:java`. For this you need Maven installed on your computer

<h3>Option 3:</h3>
<p>Run it through an IDE.</p>

<h2>Tutorial:</h2> 
<p>Click on the first level on the map and then press the handbook for instructions.</p>

<h2>Notes:</h2>
<p>If you get errors about assets not loading, try running "mvn clean" in the terminal.</p>
