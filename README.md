<h1>Rouges of Midgard</h1>
<p>To start the game, run src/main/java/org/example/DesktopLauncher.java</p>

<h2>Requirements:</h2>
<ul><li>Java 23+</li>
    <li>Maven 3.8+</li>
    <li>LibGDX 1.12.1+</li>
    <li>Gson 2.10.1+</li>
    <li>Junit 4.13.2+</li>
</ul>
    

If you get errors about assets not loading, try running "mvn clean" in the terminal

<b>Tutorial:</b> 
    Click on the first level on the map and then press the handbook for instructions.
