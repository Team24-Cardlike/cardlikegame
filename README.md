<h1>Rouges of Midgard</h1>

<h2>Requirements:</h2>
<ul>
    <li>Java 21+</li>
    <li>Maven 3.8+</li>
    <li>LibGDX 1.12.1+</li>
    <li>Gson 2.10.1+</li>
    <li>JUnit 4.13.2+</li>
</ul>

<h2>Running the game:</h2>

<h3>Option 1:</h3>
<p>Run the following command when in the directory outside the src folder (cardlikegame):</p>
<pre><code>src/main/java/org/example/DesktopLauncher.java</code></pre>

<h3>Option 2:</h3>
<p>Run the following command (requires Maven):</p>
<pre><code>mvn compile exec:java</code></pre>

<h3>Option 3:</h3>
<p>Run it through an IDE.</p>

<h2>Tutorial:</h2>
<p>Click on the first level on the map and then press the handbook for instructions.</p>

<h2>Notes:</h2>
<p>If you get errors about assets not loading, try running <code>mvn clean</code> in the terminal.</p>
