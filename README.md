To start the game, run src/main/java/org/example/DesktopLauncher.java

Requirements:
    Java 23+
    Maven 3.8+
    LibGDX 1.12.1+
    Gson 2.10.1+
    Junit 4.13.2+

If you get errors about assets not loading, try running "mvn clean" in the terminal

Tutorial: 
    Click on the first level on the map and then press on the handbook for instructions.
